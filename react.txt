Tracker React 플러그인

1. 플러그인 저장소
https://github.com/WisetrackerTechteam/Tracker-react-android-plugin

2. 플러그인 동작 방식
- 고객사의 react 안드로이드 프로젝트에서 와이즈트래커가 제공하는 플러그인 파일들을 삽입하여 react <-> Native SDK 간 데이터 연동
- react 환경에서 전달 받은 데이터를 플러그인 WiseTrackerManager.java 통해 Native SDK 호출

3. 플러그인 제공 파일

1) AndroidManifest.xml
- react 안드로이드 프로젝트의 AndroidManifest.xml 파일에 포함될 사용자 정의 설정 내용

2) build.gradle
플러그인에서 사용할 라이브러리 설정 파일 (Tracker SDK 의존성)

3) .java
플러그인 소스코드

4. 플러그인 추출 방법
플러그인 수정시 git remote push
